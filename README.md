# Meta Front-End Capstone: Restaurant Booking App

This project is a React-based front-end capstone for Meta, featuring a restaurant booking application.

## Features
- View restaurant information, highlights, and testimonials
- Reserve a table through a dynamic booking form
- Booking confirmation page
- Responsive layouts and modern UI
- Navigation using React Router
- Booking times managed with React Context and Reducer
- Custom CSS styling and interactive elements

## Technologies Used
- React
- React Router
- Context API & useReducer
- CSS
- Jest & React Testing Library (for tests)

## Getting Started
1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm start
   ```
3. Run tests:
   ```bash
   npm run test
   ```

## Project Structure
- `src/components/` - UI components (Header, Hero, Card, etc.)
- `src/pages/` - Main pages (Booking, BookingConfirmed, Main)
- `src/assets/` - Images, icons, logos
- `public/` - Static files

## Booking Flow
1. User visits the home page and views restaurant details.
2. User clicks 'Reserve a Table' and fills out the booking form.
3. Available times are dynamically managed based on date selection.
4. Upon submission, booking is confirmed and user sees a confirmation page.
